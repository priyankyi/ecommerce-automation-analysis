# Flipkart Troubleshooting

Use this page when a refresh or check gives a warning.

- If script says `failed_step null` but warnings exist, this is acceptable.
- If Google Sheets `429` happens, wait 5 minutes.
- If a credentials error happens, do not paste credentials in chat.
- If Google Ads API says `DEVELOPER_TOKEN_NOT_APPROVED`, wait for Basic Access.
- If visual search says `NEEDS_CREDENTIALS`, check `credentials/visual_search.env`.
- If `Product_Image_URL` is blank, manually fill a public image URL.
- If the safety checker fails, do not commit.
